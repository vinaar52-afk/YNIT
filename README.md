# YNIT - Next.js 14 Template

Layanan profesional untuk kebutuhan Anda.

## Fitur

- ✅ Next.js 14 dengan App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Struktur folder yang rapi
- ✅ Responsive design
- ✅ Clean white layout

## Struktur Folder

```
/app               - Halaman aplikasi
/components        - Komponen reusable
/data              - Data placeholder
/lib               - Utility functions
/styles            - Global styles
/public            - Static assets
```

## Memulai

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser ke [http://localhost:3000](http://localhost:3000)

## Perintah

- `npm run dev` - Start development server
- `npm run build` - Build untuk production
- `npm start` - Start production server
- `npm run lint` - Jalankan ESLint

## Rutas

- `/` - Homepage
- `/tour` - Halaman Tour
- `/tour/[slug]` - Detail Tour
- `/dokumen` - Halaman Dokumen
- `/dokumen/[slug]` - Detail Dokumen
- `/study` - Halaman Study
- `/study/[slug]` - Detail Study
- `/medical` - Halaman Medical
- `/medical/[slug]` - Detail Medical
- `/booking` - Halaman Booking

## Customization

Semua komponen dapat disesuaikan di `/components`. Gunakan Tailwind CSS untuk styling.
